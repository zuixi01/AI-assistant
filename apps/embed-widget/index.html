AI 智能客服 - 嵌入式聊天组件

这是一个可嵌入任意网页的 AI 聊天组件。点击右下角的聊天按钮开始体验。

使用方法

<script src="ai-chat-widget.umd.js"></script> <script> AIChatWidget.init({ apiBaseUrl: 'http://localhost:4000', tenantId: 'demo', position: 'bottom-right', primaryColor: '#2563eb' }); </script>

配置参数

  • apiBaseUrl - 后端 API 地址
  • tenantId - 租户 ID
  • position - 位置：bottom-right（默认）或 bottom-left
  • primaryColor - 主题色，默认 #2563eb

功能特性

  • AI 智能问答
  • 知识库检索
  • 多轮对话
  • 流式响应
  • 响应式设计
